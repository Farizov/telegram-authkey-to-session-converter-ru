# HC2S Converter – Telegram **AuthKey / HEX:DC → SESSION+JSON или TDATA**
> Универсальный инструмент, который преобразует любой ключ Telegram (AuthKey или HEX:DC) в файл *Session+JSON* для Telethon или в формат *TDATA*.

---
## 🌐 Доступно на нескольких языках
- Английский – эта страница
- Русский – [HC2S Converter RU](https://github.com/telegram-prime/telegram-authkey-to-session-converter-RU)
- Китайский – [HC2S Converter CN](https://github.com/telegram-prime/telegram-authkey-to-session-converter-CN)

---
## ⚙️ Основные возможности
| Возможность              | Описание                                                                 |
|--------------------------|--------------------------------------------------------------------------|
| **Принимает все типы ключей** | AuthKey, HEX:DC, DC:HEX                                                |
| **Форматы на выходе**    | Telethon *Session+JSON* или *TDATA*                                      |
| **Многопоточность**      | Параллельная обработка нескольких файлов для быстрой массовой конвертации |
| **Гибкая работа с сессиями** | Сохранять текущую сессию или выполнять вход в новую                     |
| **Холодная конвертация** | Возможность конвертировать SESSION+JSON без подключения к аккаунту      |
| **Совместимость со старыми слоями** | Опция установки совместимости со старым слоем Telegram или сохранение нового |
| **Управление сессиями**  | Опционально завершать все остальные активные сессии при конвертации     |
| **Настройка подключения**| Настраиваемое количество попыток и задержка                             |
| **Автоматизация 2FA**    | Один код для всех аккаунтов вводится вручную                             |
| **Эмуляция устройств**   | Android, iOS, desktop – старые и новые сессии                            |
| **Поддержка прокси**     | HTTP/SOCKS5, список в несколько строк; каждый поток может использовать свой прокси |
| **Сортировка результатов**| Valid / invalid / timed-out / bad-2FA → отдельные папки                 |
| **Резервное копирование исходников** | Исходные файлы копируются перед обработкой                            |
| **Полное логирование**   | Окно «Program Console Log» отображает прогресс и ошибки                 |
